Launcher Content API

Этот репозиторий развёрнут на GitHub Pages и отдает статический snapshot для лаунчера. Установите в клиенте content_api_base_url=https://<ваш-user>.github.io/<repo>/api/v1/, после чего лаунчер будет читать JSON напрямую с Pages.

Полезные ссылки

  • /api/v1/dashboard
  • /api/v1/news/patch-030
  • /api/v1/news/devstream-november
  • /api/v1/news/event-synthwave

При необходимости просто замените JSON в соответствующих файлах и запушьте изменения — GitHub Pages мгновенно обновит API-слой без серверной логики!
